CSCI 592: Using Imaging to Solve Biofilm Research Questions (SP22)

About

Description

In this course, I will first gain a basic knowledge of biofilms and biofilm imaging by reading papers. Next, I will work with multiple labs in the CBE to learn about the problems they are trying to solve and how they use imaging to do so. Then I will look into image file types and specifics of different analyses used.

Old Objectives

  • Be able to describe what a biofilm is and why they are important to understand/study
  • Be able to describe different problems biofilm researchers are trying to solve
  • Be able to describe how imaging techniques are being used to solve said problems
  • Be able to list and have a basic understanding of other analysis techniques that are used alongside imaging and how they may enhance what is learned from image data
  • Understand how different analysis/image modifications affect the conclusions that can be drawn from images

Objectives

  • Compare/contrast different image analysis techniques and in which cases they make sense.
    • Why might these techniques be used in cases where they aren't ideal?
    • How do file types limit analysis that can be run?
  • Create a well-formed general or central problem statement and options for solutions
  • Create a mind map of relationships between different biofilm problems and analysis techniques (primarily image based but include others and how they supplement image analysis)
